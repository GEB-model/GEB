# Hazards

The hazards module simulates natural hazards such as floods and windstorms.

## Available Hazards

- **[Floods](floods.md)**: Pluvial, fluvial, and coastal flood modeling using SFINCS
- **[Windstorms](windstorms.md)**: Wind-based hazard modeling

## General methods for hazards

- **[Extreme Value Analysis](../advanced_topics/extreme_value_analysis.md)**: Statistical method for estimating probabilistic return periods using GPD-POT
