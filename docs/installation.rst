Installation
#############

GEB runs on Python 3.9+ and requires the following packages to be installed.

Packages
**********
GEB requires the following packages

Required
==========
- cwatm_abm
- polygene
- xarray
- numba
- tbb
- gdal
- netcdf4
- pandas
- openpyxl
- geopandas
- numpy
- cartopy
- tornado
- python-dateutil
- rasterio
- scipy
- pyyaml

Optional
===========
- cupy

Preprocessing
================
- selenium
- chromedriver_autoinstaller
- arcgis2geojson
- pyflwdir

Plotting
==========
- matplotlib

Documentation
==============
- sphinx
- sphinx-autodoc-typehints
